## ☆gitの基本を身につける
・gitデータ管理方法<br>
・gitの本質はデータを圧縮してスナップショットで保存している<br>
・`~$ git init`<br>
・`~$ git clone <リポジトリ名>`<br>
・`~$ git add <ファイル名 or ディレクトリ名>`<br>
・`~$ git commit`<br>
・`~$ git status`<br>
・`~$ git diff`<br>
・`~$ git log`<br>
・`~$ git rm`v
・`~$ git mv`<br>
