# 【わかったこと】
・Rubyは「p」の後に半角スペース、その後に表示させたいものを記述することでターミナルに表示させられる。<br>
⚫︎演算子について<br>
・算術演算子(+,-,*,/,%)<br>
・関係演算子(>,<,<=,>=,==,!=)<br>
・論理演算子(&&,||)<br>
・代入演算子(=)<br>
・複合代入演算子:足し算、引き算などと組み合わせて代入する演算子。<br>
例）zにyを足してzに代入<br>
`z += y`<br>
⚫︎条件分岐<br>
・if文<br>
※if文では条件式のあとに改行がある場合はthenを省略できる。<br>
例）<br>
```
if 条件 then
  条件を満たした時の処理
end

if 条件A
  条件Aを満たした時の処理
elsif 条件式B
  条件式Bを満たした時の処理
else
  条件を満たさないときの処理
end
```
